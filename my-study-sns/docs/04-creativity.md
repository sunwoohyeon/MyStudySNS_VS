## 4.1 프로젝트 목적 측면의 창의성

### 기존 서비스의 한계

| 기존 서비스 | 한계점 |
|------------|--------|
| Google Keep / Notion | 손글씨 노트 인식 기능 없음 |
| 네이버 OCR | 단순 텍스트 추출만 지원, 마크다운/수식 미지원 |
| 일반 SNS (인스타, 트위터) | 학습 목적에 특화되지 않음 |
| 기존 스터디 앱 | AI 기반 자동화 기능 부재 |

### My Study SNS의 혁신적 접근

#### 1. AI 기반 학습 콘텐츠 자동화

```
기존: 사용자가 수동으로 모든 콘텐츠 입력
혁신: AI가 이미지를 분석하여 자동으로 디지털 콘텐츠 생성
```

**적용 기술**
- Google Gemini 2.5 Pro/Flash 멀티모달 AI
- 이미지 → 텍스트 → 마크다운 자동 변환
- LaTeX 수식 자동 인식

#### 2. 학습과 커뮤니티의 융합

```
기존: 개인 학습 도구 vs 소셜 네트워크 분리
혁신: AI 학습 도구 + 지식 공유 커뮤니티 통합
```

**융합 포인트**
- AI가 생성한 지식카드를 커뮤니티에 공유
- 다른 사용자의 노트에서 학습
- 해시태그 기반 학습 자료 탐색

#### 창의적 설계 포인트

| 설계 요소 | 기존 방식 | My Study SNS |
|----------|----------|--------------|
| 처리 단계 | 단일 단계 | 2단계 (추출 + 교정) |

```typescript
// 신뢰도에 따른 동적 처리
```

## 4.3 제작 및 테스트 측면의 창의성

### 다중 이미지 페이지 병합 시스템

여러 장의 노트 사진을 하나의 통합된 문서로 자동 병합하는 기능을 구현했습니다.

#### 병합 알고리즘

```typescript
// 다중 이미지 결과 통합
const mergeResults = (results: AnalysisResult[]) => {
  // 제목: 첫 번째 이미지의 제목 사용
  const title = results[0].title;

  // 내용: 페이지별로 구분하여 병합
  const content = results.map((r, i) =>
    `## 📄 페이지 ${i + 1}\n\n${r.content}`
  ).join('\n\n---\n\n');

  // 해시태그: 모든 이미지의 태그 합집합 (중복 제거)
  const hashtags = [...new Set(
    results.flatMap(r => r.hashtags)
  )].slice(0, 10);

  // 교정 정보: 모든 교정 사항 통합
  const corrections = results.flatMap(r =>
    r.refinement?.corrections || []
  ).slice(0, 15);

  // 신뢰도: 평균 계산
  const avgConfidence = results.reduce(
    (sum, r) => sum + r.confidence, 0
  ) / results.length;

  return { title, content, hashtags, corrections, avgConfidence };
};
```

#### 사용자 경험 혁신

```
기존 방식:
  1. 이미지 1 업로드 → 분석 → 저장
  2. 이미지 2 업로드 → 분석 → 저장
  3. 이미지 3 업로드 → 분석 → 저장
  → 3개의 별도 게시글 생성

My Study SNS 방식:
  1. 이미지 1, 2, 3 동시 업로드
  2. 순차 분석 (진행 상태 표시)
  3. 자동 통합
  4. 한 번에 저장
  → 1개의 통합 게시글 생성
```

### 시간표 자동 감지 시스템

다양한 시간표 형식을 자동으로 감지하고 정규화하는 시스템을 구현했습니다.

#### 지원 형식

| 형식 | 예시 | 자동 변환 |
|------|------|----------|
| 24시간제 | 9, 10, 11, 12, 13, 14 | 그대로 사용 |
| 12시간제 | 9, 10, 11, 12, 1, 2 | 13:00, 14:00으로 변환 |
| 교시제 | 1교시, 2교시, 3교시 | 설정된 시작 시간 기준 변환 |

#### 스마트 변환 로직

```typescript
// 12시간제 감지 및 변환
const convertTo24Hour = (hour: number, isPM: boolean): number => {
  if (hour === 12) {
    return isPM ? 12 : 0;
  }
  return isPM ? hour + 12 : hour;
};

// 교시제 변환 (예: 1교시 = 09:00, 50분 수업 + 10분 휴식)
const periodToTime = (period: number, firstPeriodStart: string): string => {
  const [startHour, startMin] = firstPeriodStart.split(':').map(Number);
  const totalMinutes = startHour * 60 + startMin + (period - 1) * 60;
  const hour = Math.floor(totalMinutes / 60);
  const min = totalMinutes % 60;
  return `${hour.toString().padStart(2, '0')}:${min.toString().padStart(2, '0')}`;
};
```

### 충돌 검사 시스템

기존 시간표와의 시간 충돌을 자동으로 감지합니다.

```typescript
// 시간 겹침 검사
const checkTimeOverlap = (
  start1: string, end1: string,
  start2: string, end2: string
): 'full' | 'partial' | null => {
  const [s1, e1] = [toMinutes(start1), toMinutes(end1)];
  const [s2, e2] = [toMinutes(start2), toMinutes(end2)];

  if (s1 >= s2 && e1 <= e2) return 'full';    // 완전 포함
  if (s1 < e2 && e1 > s2) return 'partial';   // 부분 겹침
  return null;                                  // 겹침 없음
};
```

## 4.6 결론

My Study SNS는 다음과 같은 창의적 요소를 통해 기존 서비스와 차별화됩니다:

1. **2단계 AI OCR 시스템**: 신뢰도 기반 선택적 교정으로 정확도와 효율성 동시 확보
2. **다중 이미지 병합**: 여러 장의 노트를 하나의 문서로 자동 통합
3. **시간표 자동 인식**: 다양한 형식의 시간표를 자동 감지 및 정규화
4. **교정 투명성**: AI 교정 과정을 사용자에게 투명하게 공개
5. **학습-커뮤니티 융합**: AI 도구와 소셜 기능의 유기적 결합

이러한 창의적 접근을 통해 단순한 OCR 도구를 넘어 **AI 기반 학습 생태계**를 구축했습니다.
