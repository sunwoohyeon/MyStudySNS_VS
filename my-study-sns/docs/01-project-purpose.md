# 1. 프로젝트 목적 (20점)

## 1.1 프로젝트 개요

### 프로젝트명
**My Study SNS** - AI 기반 스터디 커뮤니티 플랫폼

### 프로젝트 요약
My Study SNS는 학생들의 학습 효율성을 극대화하기 위한 AI 기반 스터디 커뮤니티 플랫폼입니다. Google Gemini AI를 활용하여 손글씨 노트를 자동으로 디지털화하고, 시간표 이미지를 분석하며, 학습 콘텐츠를 지식 카드로 변환하는 혁신적인 기능을 제공합니다.

---

## 1.2 캡스톤디자인 주제 부합성

### 주제 선정 배경

| 구분 | 내용 |
|------|------|
| **사회적 배경** | 디지털 학습 환경의 급격한 변화와 AI 기술의 발전 |
| **기술적 배경** | 멀티모달 AI(Gemini)의 등장으로 이미지-텍스트 처리 가능 |
| **교육적 배경** | 학생들의 비효율적인 노트 관리 및 학습 자료 공유의 어려움 |

### 캡스톤디자인 적합성

1. **실용적 문제 해결**
   - 학생들이 실제로 경험하는 "손글씨 노트 디지털화" 문제를 AI로 해결
   - 시간표 이미지를 찍으면 자동으로 일정 등록

2. **최신 기술 활용**
   - Google Gemini 2.5 Pro/Flash AI 모델 적용
   - Next.js 15, Supabase 등 현대적 웹 기술 스택

3. **산학 연계 가능성**
   - 에듀테크 기업과의 협업 가능
   - 대학교 학습 관리 시스템(LMS)과의 통합 가능

## 1.3 명확하고 집중된 프로젝트 목적

### 핵심 목표

```
"AI를 활용하여 학생들의 학습 콘텐츠를 디지털화하고,
커뮤니티를 통해 지식을 공유하는 플랫폼 구축"
```

### 세부 목표

| 번호 | 목표 | 구현 방법 |
|------|------|-----------|
| 1 | 손글씨 노트 디지털화 | Gemini OCR + 2단계 AI 교정 시스템 |
| 2 | 시간표 자동 인식 | 이미지 분석을 통한 수업 정보 추출 |
| 3 | 지식 카드 자동 생성 | 게시글 분석 → 학습용 플래시카드 변환 |
| 4 | 학습 커뮤니티 구축 | 게시판, 해시태그, 추천 시스템 |

### 프로젝트 범위

**포함 범위 (In Scope)**
- AI 기반 노트 OCR 및 교정
- 시간표 이미지 분석 및 자동 등록
- 지식 카드 자동 생성
- 게시판 기반 커뮤니티 (자유게시판, 질문/답변, 스터디노트)
- 사용자 인증 (이메일, Google, GitHub, Kakao)
- 반응형 웹 UI (라이트/다크 모드)

**제외 범위 (Out of Scope)**
- 모바일 네이티브 앱
- 실시간 채팅/화상 스터디
- 유료 결제 시스템

## 1.5 과학적으로 검증 가능한 접근 방법

### AI 시스템 설계 철학

#### 2단계 OCR 시스템

```
┌─────────────────────────────────────────────────────────────────┐
│                     Phase 1: 초기 추출                          │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │  Gemini 2.5 Pro를 사용한 이미지 OCR                      │   │
│  │  - 손글씨/인쇄물 판별                                    │   │
│  │  - 마크다운 형식으로 변환                                │   │
│  │  - LaTeX 수식 인식                                       │   │
│  │  - 신뢰도 점수 산출 (0.0 ~ 1.0)                         │   │
│  └─────────────────────────────────────────────────────────┘   │
│                              ↓                                  │
│                    신뢰도 < 0.9 인 경우                         │
│                              ↓                                  │
│                     Phase 2: AI 교정                            │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │  Gemini 2.5 Pro를 사용한 텍스트 교정                     │   │
│  │  - OCR 오류 수정 (예: "톰" → "톰슨")                     │   │
│  │  - 전문 용어 정확성 검증                                 │   │
│  │  - 문맥 보완                                             │   │
│  │  - 교정 내역 기록                                        │   │
│  └─────────────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────────────┘
```

#### 검증 가능한 메트릭

| 메트릭 | 측정 방법 | 목표값 |
|--------|----------|--------|
| **OCR 정확도** | 원본 대비 인식률 | 95% 이상 |
| **신뢰도 점수** | AI 자체 평가 | 0.85 이상 |
| **교정 정확도** | 오류 수정률 | 90% 이상 |
| **처리 시간** | 이미지당 분석 시간 | 10초 이내 |

### 기술적 검증 방법

1. **단위 테스트**
   - 다양한 이미지 형식 (JPEG, PNG, WebP)
   - 손글씨 vs 인쇄물 분류 정확도
   - 수식 인식 테스트

2. **통합 테스트**
   - 이미지 업로드 → OCR → 저장 전체 흐름
   - 에러 핸들링 검증

3. **사용자 테스트**
   - 실제 노트 이미지로 테스트
   - 결과물 만족도 조사

## 1.7 프로젝트 차별점

### 기존 서비스 vs My Study SNS

| 비교 항목 | 기존 OCR 서비스 | My Study SNS |
|----------|----------------|--------------|
| OCR 방식 | 단순 텍스트 추출 | 2단계 추출 + AI 교정 |
| 출력 형식 | Plain Text | 마크다운 (수식, 코드 지원) |
| 교정 기능 | 없음 | AI 기반 자동 교정 |
| 커뮤니티 | 없음 | 학습 커뮤니티 통합 |
| 지식 카드 | 없음 | AI 자동 생성 |
| 시간표 | 없음 | 이미지 자동 분석 |

### 핵심 차별화 기술

1. **신뢰도 기반 선택적 교정**
   - 신뢰도 0.9 이상: 교정 생략 (성능 최적화)
   - 신뢰도 0.9 미만: 자동 교정 수행

2. **다중 이미지 페이지 병합**
   - 여러 장의 노트 사진을 하나의 문서로 자동 통합
   - 페이지별 구분자 자동 삽입

3. **LaTeX 수식 완벽 지원**
   - 수학/과학 노트의 수식을 LaTeX로 변환
   - KaTeX 렌더링으로 즉시 미리보기
